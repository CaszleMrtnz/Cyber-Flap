🕹️ CYBER FLAP
A Neon Reflex Challenge Developed by Jade Caszle S. Martinez (CPGSHS) Under the advisory of Mr. Gerald Tuyogon

🚀 Project Overview
Cyber Flap is a high-speed reflex game where you control the "Cyber-Pulse." Your mission is to navigate through neon-lit security gates without crashing. This project focuses on responsive web design and real-time game logic using HTML5 Canvas.

⚡ Key Features
Level Evolution: The city environment and colors change every 10 points.
Neon UI: A clean, futuristic interface designed for high visibility.
Smooth Controls: Optimized for both keyboard and touch-screen play.
Live Feedback: Integrated system for direct email suggestions.

🎮 How to Play:
Action              Control
Jump                FlySpacebar or Tap Screen
Pause Game          P Key
Restart             Spacebar (on Game Over screen)

⚠️ Developer’s Note:
This system is currently in active development. I am working on adding more "Sectors" (levels) and improving the physics of the Cyber-Pulse.

Feedback is highly appreciated! Please use the in-game feedback button to help me improve the system.
